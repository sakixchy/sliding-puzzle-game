Sliding Puzzle

How To Play?


Simply drag a tile and swap it with another one adjacent to it.
You can only move a tile with an empty (white tile),
Try to rearrange all the pieces within the limit of the timer.
The Timer starts as soon as you make your first move!
All the best!

Features:

  • Pause function
  • Background Music
  • Highscore Table
